# ドラッグ＆ドロップコメント
ニコニコ動画の動画ページ上で定型コメントをコメント入力欄へドラッグ＆ドロップすることでコメントをできるようにする Chrome 拡張機能です

## インストール方法
[Chrome ウェブストア](https://chromewebstore.google.com/detail/%E3%83%89%E3%83%A9%E3%83%83%E3%82%B0%EF%BC%86%E3%83%89%E3%83%AD%E3%83%83%E3%83%97%E3%82%B3%E3%83%A1%E3%83%B3%E3%83%88/oigjfdodpifhfnnhglknfldbjhgmnfme)からインストールできます

## 使い方
ニコニコ動画の動画ページ上にあるコメント入力欄の下に表示されたテキストをコメント入力欄にドラッグ＆ドロップするとコメントが送信されます
連投防止のため、一度ドラッグ＆ドロップでコメントを送信したあと一定時間ドラッグ＆ドロップができなくなっています

## おことわり
- 本拡張機能についてニコニコおよび株式会社ドワンゴへのお問い合わせはおやめください
- 本拡張機能のインストール後、ニコニコ動画の動作が正常でなくなった場合は使用を中止してください
- ニコニコ動画のアップデートにより本拡張機能が動作しなくなる場合がございます
- 本拡張機能の継続的なアップデートはお約束できません
- 本拡張機能を利用して生じたいかなる損害や損失、トラブルの責任を本拡張機能の作成者は一切負いません

## なぜ作ったのか
むかしのニコニコ動画にあった「かんたんコメント機能」のような機能がほしかったので作りました
本当はボタンをクリックすることでコメントを送信するようにしたかったのですが、Reactをハックするようなものを書かないといけなさそうだったので、よりお手軽な実装としてドラッグ＆ドロップでコメント送信するというものになりました
